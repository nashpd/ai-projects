# Google Sheet Template

Open the template here: [Eisenhower Task Manager — Google Sheet](https://docs.google.com/spreadsheets/d/1CjkV3VCkCoYdiMAaJoyTcbodnwSywy3HvqkXRmOx0wA/copy)

To use it: open the link, go to File > Make a Copy, and save it to your Google Drive. The copy is yours to edit. Nothing you do will affect the original.

The sheet has six columns ready to go:

| Column | Purpose |
|---|---|
| A | Task — write your tasks here |
| B | Deadline — if there is one |
| C | Urgency — AI fills this: High or Low |
| D | Importance — AI fills this: High or Low |
| E | Quadrant — AI fills this: Q1, Q2, Q3, or Q4 |
| F | Action — you write one next step per task |

Use the prompts in `prompt-library.md` to populate columns C, D, and E.
