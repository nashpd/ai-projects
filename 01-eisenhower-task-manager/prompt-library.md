# Prompt Library — Eisenhower Task Manager

Copy and paste these prompts directly into Claude or ChatGPT.

---

## Prompt 1: Initial Task Sort

Use this to sort a new list of tasks.

```
I am going to give you a list of tasks. For each task, I need you to help me sort it using the Eisenhower Matrix. The four categories are:

1. Urgent and Important: do it now
2. Important but Not Urgent: schedule it
3. Urgent but Not Important: delegate it
4. Neither Urgent nor Important: drop it

Before you sort each task, ask me two questions if you are not sure: What happens if this does not get done today? And does this directly affect a key goal or commitment I have?

Here are my tasks:
[paste your task list here]
```

---

## Prompt 2: Weekly Refresh

Use this at the start of each week to add new tasks to an existing sorted list.

```
Here is my existing sorted task list from last week:
[paste your current sheet contents here]

I have new tasks to add this week:
[paste new tasks here]

Please sort the new tasks using the Eisenhower Matrix and tell me if any of the existing tasks should be reclassified based on what has changed.
```

---

## Prompt 3: Single Task Clarity

Use this when you are unsure how to classify one specific task.

```
I have one task I cannot decide how to classify using the Eisenhower Matrix: [describe the task].

Here is the context: [explain what it is, who it involves, and what happens if it is delayed].

Which quadrant does this belong in, and why?
```

---

## Prompt 4: Stuck on a Quadrant 1 List That Is Too Long

Use this when everything feels urgent and the Q1 list is unmanageable.

```
My Quadrant 1 list has too many items and I cannot work through all of them today. Here they are:
[paste your Q1 list]

Help me identify which 3 of these are genuinely Quadrant 1 and which ones I may have misclassified. Ask me clarifying questions if needed.
```
